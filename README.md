## Introduction
A .NET library that provides a stepwise log view control for WPF.  
Package Id: `QIQI.WpfStepwiseLog`

## Build
```bash
dotnet restore
dotnet build --no-restore
dotnet pack --no-build
```
or in short:
```bash
dotnet pack
```
